# Toimiala-AI-skannaus — Systemaattinen markkina-aukkojen tunnistus

Metodologia AI-agentti-mahdollisuuksien löytämiseen toimiala kerrallaan. Tulos: 3–5 konkreettista kaupallistettavaa AI-agenttiideaa todistein.

## Prosessi (4 vaihetta)

### Vaihe 1: Rajaus

Ennen skannausta määrittele:
- **Toimiala**: Esim. logistiikka, rakennus, terveydenhuolto, koulutus
- **Kohdeasiakas**: PK-yritys, suuryritys, julkishallinto
- **Maantiede**: Suomi / Pohjoismaat / EU
- **Fokus-osa-alue**: Mitkä toiminnot ovat kriittisiä?

### Vaihe 2: Kipupisteiden kartoitus (min. 7 ongelmaa)

Tee vähintään 7 hakua — älä spekuloi, pohjaa todisteisiin:

```
"[ala] operational challenges AI automation 2026"
"[ala] pain points small business Finland"
"[ala] manual processes inefficiency cost"
"[ala] regulatory compliance burden SME"
"[ala] labor shortage automation opportunity"
"[ala] AI adoption barriers Nordic Europe"
"[ala] software market gaps underserved segment"
```

Dokumentoi jokaisesta ongelmasta: kuvaus, kuka kärsii, miksi ratkaisematta.

### Vaihe 3: Markkina-aukkojen tunnistus

Jokaiselle kipupisteelle tee 3 kilpailijahakua:

```
"[ongelma] AI solution startup 2026"
"[ongelma] software Finland OR Nordic"
"[ongelma] SaaS pricing small business"
```

**Luokittelu:**
- 🔴 Ei ratkaisua → Korkea prioriteetti
- 🟡 Ratkaisu olemassa, mutta kallis/globaali/monimutkainen → Mahdollisuus
- 🟢 Hyvin katettu → Ohita

### Vaihe 4: Agenttiideoiden muotoilu ja pisteytys

Top 3–5 aukolle muotoile AI-agenttiidea ja pistytä:

| Kriteeri | Pisteet (0–3) |
|----------|---------------|
| Kipupisteen vakavuus | Kriittinen=3, Tärkeä=2, Kiva=1 |
| Kilpailutilanne | Tyhjä=3, Heikot=2, Vahvat=1 |
| Tekninen toteutettavuus | Helppo=3, Kohtalainen=2, Vaikea=1 |
| PK-kattavuus Suomessa | Ei kukaan=3, Osittain=2, Kyllä=1 |

**Agenttikortti per idea:**

```
Nimi: [Selkeä tuotenimi]
Ongelma: [1 lause — mitä kivistää]
Ratkaisu: [1 lause — miten AI ratkaisee]
Asiakas: [Kuka maksaa ja miksi nyt]
Kilpailutilanne: [Tyhjiö / Heikot kilpailijat — perustelu]
ROI-arvoväite: [Konkreettinen säästö tai hyöty euroina tai %]
Pisteet: [X/12]
```

## Raporttipohja

```
# [Toimiala] AI-skannaus — [päivämäärä]

## Tunnistetut kipupisteet (7+)
[Lista ongelmista viitteineen]

## Markkina-aukot (luokiteltu)
[🔴 Korkea / 🟡 Mahdollisuus / 🟢 Ohita]

## Top AI-agenttiideat
[3–5 agenttikorttia pisteytyksineen]

## Suositus: Paras mahdollisuus
[Perusteltu valinta — miksi juuri tämä, miksi nyt]
```

## Muistisäännöt

- Hae aina 7+ ongelmaa ENNEN kuin arvioit kilpailua
- Tarkista aina: onko kansainvälinen ratkaisu saatavilla suomalaisille PK-yrityksille?
- Konkretia: euromääräinen ongelma tai mitattava tehottomuus on vahvempi kuin epämääräinen "haaste"
- 2026: Agentic AI -markkina on $7–15B ja vasta 24% B2B-yrityksistä on ottanut käyttöön — markkinapotentiaali on realistinen
